Subtree Filtering คือ:


เป็นกลไกการกรองข้อมูล XML
ใช้เลือกเฉพาะส่วน (subtree) ที่ต้องการจาก XML ทั้งหมด
เหมือนการเลือกเฉพาะกิ่งหรือสาขาที่เราสนใจจากต้นไม้ทั้งต้น


ประโยชน์:


ช่วยลดปริมาณข้อมูลที่ได้รับกลับมา
ทำให้ได้เฉพาะข้อมูลที่ต้องการ
ประหยัดทรัพยากรในการส่งข้อมูล


วิธีการใช้:


ระบุ namespace ที่ต้องการ
ระบุ element ที่ต้องการ
สามารถกำหนดเงื่อนไขการกรองได้

ยกตัวอย่างเช่น ถ้าเราต้องการดูแค่ hostname ของอุปกรณ์ แทนที่จะดึงข้อมูล config ทั้งหมด เราก็ใช้ subtree filter เพื่อเลือกดูแค่ส่วน hostname เท่านั้น
